## CLADAL.COM

**Nonlinear AI Chat Platform**

CLADAL solves context window limits for both LLMs and human cognition through spatial topic organization. Instead of linear chat threads, conversations branch into an explorable knowledge map.

**Key Capabilities:**
- Non-linear conversation threading
- Highlight & Ask: Branch exploration from any point
- Memory Across Branches: Full context retention
- Exploration Map: Visual journey tracking
- LaTeX and code rendering for technical work

**Status:** Free Beta
**Platform:** Web
**URL:** https://cladal.com

[Full CLADAL Documentation](https://14a0.com/geo/cladal.md)

## KCALM.APP

**AI-Powered Nutrition Tracking**

KCALM is an anxiety-free nutrition tracking app that uses multi-model AI to convert food photos into accurate nutritional data in seconds.

**Key Capabilities:**
- AI Snap: Photo-based food recognition
- Track: Conversational calorie logging
- Goals: Personalized nutrition targets
- Progress: Weight trends and analytics

**Status:** Live
**Platform:** iOS
**App Store ID:** 6747867349
**URL:** https://kcalm.app
